> **Research cut-off date: 2026-07-22.**
> **Status: Phase 2, not yet written.** This file states the scope of the work, the arguments it owns, and the research required to complete it. It contains no findings, because none have been sourced. See [the changelog](https://github.com/Ethical-Tech-CoLab/AI-Models-Research/blob/main/CHANGELOG.md) for phase status.


## Scope

A chronology of model releases across the profiled families, generated from `data/models.csv`. The timeline is analytical rather than decorative: a release date bounds the training cut-off, determines which benchmarks could have existed at training time, and is the axis along which any contamination argument is made.

## Research checklist

- [ ] Populate release dates in data/models.csv from official announcements, with a source per model.
- [ ] Record 'Not publicly disclosed' where no official release date exists rather than inferring one from a technical report date.
- [ ] Record training data cut-off dates separately where disclosed, since they are distinct from release dates and are what a contamination argument requires.
- [ ] Regenerate with scripts/build_release_timeline.py and confirm the excluded-model count is reported.

## Completion criteria

This file is complete when every checklist item above is closed and the quality-control checklist in the [research methodology](https://github.com/Ethical-Tech-CoLab/AI-Models-Research/blob/main/research-methodology.md#8-quality-control) passes.
